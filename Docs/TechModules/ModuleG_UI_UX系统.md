# 模块 G：UI/UX 系统 (UI/UX System)

> **技术优先级：⭐⭐⭐**
> 本模块负责所有界面层的布局、交互与数据绑定。依赖模块 A（血量/锁定数据）、模块 D（经验/计时/Boss 触发）、模块 B（三选一选项数据）、模块 E（局外天赋数据）。

---

## 一、模块介绍

本模块实现游戏内所有用户界面，包括：局内战斗 HUD、三选一升级面板、Boss 预警遮罩、剧情对话框，以及局外的主菜单、结算界面和天赋树升级面板。

整体 UI 风格定义为"近未来科幻、极简玻璃拟态（Glassmorphism）"，色彩以深邃蓝、全息青、警告红为主。剧情对话系统需要支持 2D 立绘、表情差分和打字机效果，且对话期间游戏不暂停。

---

## 二、模块职责

- **局内 HUD**：经验进度条 + 等级显示 + 倒计时；战机血条（绑定战机或屏幕中下方）
- **三选一升级面板**：触发时高斯模糊暗化背景；3 张卡牌 + 确认/刷新按钮；侧边当前武器槽位状态预览
- **Boss 预警**：全屏红边 WARNING 遮罩动画；顶部 Boss 名称 + 血条
- **剧情对话框**：屏幕底部半透明通讯面板；主角 + 39号 AI 立绘（支持多表情差分）；打字机效果；游戏不暂停；点击/自动推进
- **主菜单**：动态背景（战机停靠整备舱）+ 开始/设置/退出按钮
- **结算界面**：击杀数 + 存活时间 + 研究点数字跳动动画
- **天赋树面板**：树状/矩阵节点图；鼠标悬浮显示花费与效果；点击消耗货币解锁

## 三、对应需求原文

> 以下为《视界边缘》2.0 正式需求案原文，完整摘录，不做删减。

---

### 【需求原文】8.2 界面交互与 UI 资源 (UI/UX Design)

UI 整体采用 **"近未来科幻、极简玻璃拟态（Glassmorphism）"** 风格，色彩以深邃蓝、全息青、警告红为主。

---

#### 【UI 资源与切图清单】

**局内 HUD (Head-Up Display)：**

- **顶部信息栏：** 贯穿屏幕顶部的经验值进度条（条带发光）、当前等级数字、当前关卡存活时间倒计时。
- **战机状态栏：** 屏幕中下方（或绑定在战机模型下方），显示简易血条。
- **Boss 预警：** 全屏边缘泛红的"WARNING"遮罩动画，顶部出现巨大的 Boss 血条及名字。
- **剧情对话框：** 屏幕底部的半透明通讯面板。包含主角立绘、39 号 AI 立绘、打字机效果的文本显示区。

**局内升级面板 (Roguelite 三选一界面)：**

- **背景：** 触发时游戏画面暂停并高斯模糊变暗。
- **卡牌底框：** 3 张科技感卡牌（包含：武器/装备 Icon 区、名称、稀有度边框、描述文本区、流派词条 Tag 小图标）。
- **交互按钮：** "确认装备"按钮、"刷新（剩余 X 次）"按钮。
- **状态预览面板：** 屏幕侧边显示玩家当前已装备的 4 个武器槽位状态，提示玩家"若选择新武器将替换哪一个"。

**局外系统面板 (Meta 界面)：**

- **主界面 (Main Menu)：** 动态背景（战机停靠在太空基地的整备舱），包含 开始游戏、设置、退出按钮。
- **结算界面：** 战斗总结，显示击杀数、存活时间、转化为"研究点数"的数字跳动结算动画。
- **天赋树/升级面板：** 树状或矩阵排列的科技节点图标（血量、移速、槽位等），鼠标悬浮显示升级花费与效果。

---

### 【需求原文】7.3.1 演出表现形式（对话 UI 规格）

- **触发时机：** 进入新关卡，Boss 战之前。

- **UI 表现：** 画面底部弹出半透明科技风对话框。左侧是主角戴着头盔的 2D 立绘（有不同表情差分，如：懵逼、愤怒、得意），右侧是 39 号的立绘。

- **交互方式：** 局内对话时游戏不暂停，玩家可继续移动战机，对话文本自动播放或点击鼠标推进，类似街机 STG 的过场通讯。

---

### 【需求原文】5.1.2（节选）Boss 预警 UI 规格

**Boss 登场与战斗激活：** 出现全屏警告的同时 Boss 从屏幕外飞入场，同时将玩家的飞机强制移到屏幕下方，防止双方撞到。警告结束后，**屏幕上方显示 boss 的名字和血条**，Boss 战正式开始。

---

### 【需求原文】8.1（节选）锁定 UI 表现（含于 HUD 规格）

**视界范围显示（核心：如何优雅地显示在屏幕上而不光污染？锁定的 UI 表现是什么？）**

- **进度积累状态：** 敌人脚下或身上出现瞄准镜的图案，开始全是暗红色的，但随着时间顺时针填充为亮红色，代表锁定完成。
- **锁定完成瞬间：** 准星瞬间缩紧，伴随一次极短暂的高亮闪烁。

---

### 【需求原文】3.2（节选）三选一界面交互逻辑

当战机升级时，游戏画面暂停，弹出"局内升级面板"。

- **选项池：** 系统会从当前解锁的"武器库"和"装备库"中随机抽取 3 个选项供玩家选择。
- **刷新机制：** 默认有 1 次刷新机会，永久升级里可以提升到 3 次。
